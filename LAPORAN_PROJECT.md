# Laporan Project: Website Ngeselin Games (Digital Psychological Warfare)

**Judul Project:**
The Ultimate Annoying Website: A Journey of Infinite Suffering

**Deskripsi Lengkap:**
Sebuah mahakarya front-end engineering yang dirancang murni menggunakan HTML, CSS modern (Glassmorphism), dan Vanilla JavaScript tanpa campur tangan backend (Zero PHP/Database). Website ini mensimulasikan "Game Psikologis" yang menguji kesabaran pengunjung melalui puluhan teka-teki absurd dan rintangan yang melawan logika UX/UI standar.

Fitur Utama:
1. **Procedural Generation System (Roguelike):** Setiap kali user mereset permainan, sistem akan mengacak ulang jawaban, lokasi tombol, dan teka-teki menggunakan Seed Randomizer di LocalStorage.
2. **Ironclad Security:** Dilengkapi dengan Anti-Inspect Element, Anti-Right Click, dan Debugger Loop Trap yang mematikan developer tools secara paksa untuk mencegah kecurangan.
3. **Ultra-Modern UI:** Memiliki fitur Widget Translate kustom bergaya Glassmorphism yang memicu API Google Translate secara tersembunyi, mendukung multi-bahasa dengan mulus.
4. **Interactive Media:** Memiliki background music elevator yang meloop persisten, efek suara (Jumpscare/Meme), serta ditutup dengan "Fake Win" yang memicu auto-play Rickroll.

Tujuan project ini adalah sebagai sarana hiburan interaktif dan eksperimen sejauh mana JavaScript murni dapat memanipulasi DOM dan emosi pengguna dalam satu waktu yang bersamaan.

**Keterangan (Max 250 Karakter):**
Game teka-teki web interaktif tanpa backend. Memanipulasi DOM JS untuk menciptakan rintangan absurd yang diacak tiap reset. Dilengkapi UI Glassmorphism, Multi-bahasa rahasia, Anti-Hacker trap, dan ditutup dengan Rickroll. Murni front-end magic.
